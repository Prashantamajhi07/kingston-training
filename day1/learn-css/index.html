1. Image Properties

CSS provides several ways to style images:

  • max-width: Ensures the image doesn't exceed its container's width.
  • height: Auto-adjusts to maintain the aspect ratio.
  • border: Adds a border around the image.
  • border-radius: Rounds the corners or creates circular shapes.
  • box-shadow: Adds shadow effects.

2. CSS Gradients

Gradients are smooth transitions between two or more colors:

  • Linear Gradient: Transitions colors along a straight line.
  • Radial Gradient: Transitions colors outward in a circular or elliptical shape.
